# Supabase 學習手冊

作者：Joker

## 簡介：15 分鐘，從零到後端英雄

本手冊專為追求效率的您設計。我們將跳過所有冗長的理論，直接進入實戰。在接下來的 15 分鐘內，您將學會使用 Manus Supabase 連結器，完成一個應用程式後端所需的核心功能。本手冊的目標不是讓您成為理論專家，而是讓您成為一個能快速動手解決問題的實踐者。

準備好了嗎？讓我們開始這趟高速學習之旅。

## 學習目標

完成本手冊後，您將能夠：

1.  **獨立完成資料庫的 CRUD 操作**：使用自然語言查詢、新增、更新和刪除資料庫紀錄。
2.  **管理使用者身份驗證**：透過簡單指令處理使用者註冊與登入。
3.  **實作檔案上傳功能**：將本地檔案輕鬆存放到雲端。

## 關鍵技能培養

### 技能一：與資料庫對話 (CRUD)

這是您將要掌握的最核心、最頻繁使用的技能。所有動態應用程式的基礎，都建立在與資料庫的互動之上。

#### 實踐練習 1：建立您的第一個資料庫紀錄

**情境**：您正在開發一個部落格應用，需要發布第一篇文章。

**指令**：
> 「在 `posts` 資料表中新增一筆紀錄，設定 `title` 為 'My First Post'，`content` 為 'Hello, world!'。」

**練習**：請修改以上指令，將 `title` 和 `content` 換成您自己的內容，並成功執行它。確認 Manus 回傳了成功新增的訊息。

#### 實踐練習 2：查詢並過濾您的資料

**情境**：您的部落格文章越來越多，您需要找到特定的一篇。

**指令**：
> 「從 `posts` 資料表中，查詢 `title` 等於 'My First Post' 的文章。」

**練習**：嘗試不同的過濾條件。例如，如果您的資料表有 `is_published` 欄位，試著查詢所有「已發布」的文章。

#### 實踐練習 3：更新與刪除

**情境**：您發現第一篇文章有錯字，需要修改它。後來又決定刪除這篇草稿。

**指令**：
1.  > 「更新 `posts` 表，將 `title` 為 'My First Post' 的文章 `content` 更新為 'Hello, Supabase!'。」
2.  > 「從 `posts` 表中，刪除 `title` 為 'My First Post' 的文章。」

**練習**：請完整執行一次「新增 -> 查詢 -> 更新 -> 刪除」的生命週期。這個循環是所有資料庫操作的基礎。

### 技能二：管理您的使用者 (Auth)

沒有使用者系統的應用，就像一座沒有居民的城市。學會管理使用者，是讓您的應用活起來的關鍵。

#### 實踐練習 4：建立與登入使用者

**情境**：一位新使用者想要註冊您的部落格服務。

**指令**：
1.  > 「使用電子郵件 `user@example.com` 和密碼 `strong_password` 註冊一個新使用者。」
2.  > 「使用電子郵件 `user@example.com` 和密碼 `strong_password` 登入。」

**練習**：請使用您自己的信箱和一個安全的密碼，完成註冊與登入流程。觀察 Manus 回傳的使用者資訊與 session token。

### 技能三：儲存您的檔案 (Storage)

文字之外，圖片、影片和文件等豐富媒體是現代應用的標準配備。

#### 實踐練習 5：上傳您的頭像

**情境**：使用者想要上傳一張個人頭像。

**準備**：請先在您的電腦上準備一張圖片檔案，例如 `avatar.png`，並知道它的本地路徑。

**指令**：
> 「將本地檔案 `/path/to/your/avatar.png` 上傳到名為 `avatars` 的儲存桶中。」

**練習**：請將路徑替換為您自己的檔案路徑，並成功上傳。取得 Manus 回傳的公開 URL，並在瀏覽器中打開它，確認圖片可以被存取。

## 學習檢核清單

在結束這趟學習旅程之前，請對照以下清單，確認您已掌握所有關鍵技能：

-   [ ] 我能成功使用 `database.insert` 新增一筆資料庫紀錄。
-   [ ] 我能成功使用 `database.select` 搭配 `filter` 查詢到指定的資料。
-   [ ] 我能成功使用 `database.update` 修改一筆已存在的紀錄。
-   [ ] 我能成功使用 `database.delete` 刪除一筆紀錄。
-   [ ] 我能成功使用 `auth.signUp` 建立一個新的使用者帳號。
-   [ ] 我能成功使用 `auth.signInWithPassword` 登入一個已存在的帳號。
-   [ ] 我能成功使用 `storage.upload` 將一個本地檔案上傳到 Supabase。

如果您對以上所有問題的回答都是「是」，那麼恭喜您！您已經具備了使用 Manus Supabase 連結器進行快速後端開發的核心能力。現在，是時候將這些技能應用到您自己的專案中了。
