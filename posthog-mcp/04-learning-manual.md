# PostHog 連結器學習手冊

作者：Joker

---

本手冊專為希望快速上手 PostHog 連結器的使用者設計，著重於關鍵技能的培養。我們將提供清晰的學習目標和具體的實踐練習，幫助您在最短的時間內掌握最有價值的分析能力。

## 核心學習目標

完成本手冊後，您將能夠：

1.  **追蹤關鍵使用者行為**：知道如何定義和記錄對您的產品最重要的事件。
2.  **建立核心指標儀表板**：能夠將原始數據轉化為可視化的趨勢圖和漏斗圖。
3.  **執行使用者行為分析**：能夠回答「使用者從哪裡來？」、「他們在做什麼？」以及「他們為什麼離開？」等基本問題。
4.  **利用功能旗標進行測試**：學會如何安全地發布新功能並衡量其影響。

## 學習路徑與實踐練習

### 第一課：事件追蹤 (Event Tracking)

**目標**：學會使用 `capture_event` 工具記錄使用者行為。

**為什麼重要**：事件是所有分析的原子單位。沒有準確的事件追蹤，就沒有可靠的數據分析。

**實踐練習**：

1.  **定義一個關鍵事件**：思考您的產品中最重要的使用者行為是什麼？是「註冊」、「購買」，還是「發布內容」？
2.  **記錄您的第一個事件**：使用 Manus 發送一個指令來記錄這個事件。

    > **指令範例**：「幫我記錄一個 `user_signup` 事件，使用者 ID 是 `new-user-1`，註冊來源是 `google`。」

3.  **為事件添加屬性**：思考除了事件本身，還有哪些上下文資訊是重要的？例如，使用者從哪個廣告活動來的？他們使用的是什麼設備？

    > **指令範例**：「幫我記錄一個 `product_purchased` 事件，使用者 ID 是 `customer-55`，商品 ID 是 `prod-123`，金額是 `99.99`。」

### 第二課：趨勢分析 (Trend Analysis)

**目標**：學會使用 `get_insights` 將事件數據轉化為時間序列圖表。

**為什麼重要**：趨勢圖可以幫助您直觀地了解核心指標隨時間的變化，是監控產品健康狀況最直接的方式。

**實踐練習**：

1.  **選擇一個核心指標**：選擇您在第一課中定義的關鍵事件，例如 `user_signup`。
2.  **查詢每日趨勢**：使用 Manus 查詢該事件在過去 30 天的每日發生次數。

    > **指令範例**：「幫我查詢過去 30 天內，`user_signup` 事件的每日趨勢。」

3.  **進行維度下鑽**：嘗試按事件屬性（如「註冊來源」）對趨勢進行分組比較，找出增長的驅動因素。

    > **指令範例**：「幫我查詢過去 30 天內，`user_signup` 事件的每日趨勢，並按『註冊來源』屬性進行分組。」

### 第三課：漏斗分析 (Funnel Analysis)

**目標**：學會使用 `get_insights` 建立漏斗，分析使用者旅程中的轉化率。

**為什麼重要**：漏斗分析可以精確地定位使用者在完成關鍵任務時在哪個環節流失最多，是優化使用者體驗、提升轉化率的利器。

**實踐練習**：

1.  **定義一個關鍵流程**：選擇一個對您的業務至關重要的使用者流程，例如「註冊流程」、「購買流程」或「發布內容流程」。
2.  **拆解流程步驟**：將該流程拆解為 2-4 個關鍵步驟，並確保每個步驟都對應一個已追蹤的事件。
3.  **建立漏斗**：使用 Manus 建立一個漏斗分析，觀察每一步的轉化率。

    > **指令範例**：「幫我建立一個購買漏斗，步驟是 `viewed_product`、`added_to_cart`、`started_checkout`、`completed_purchase`。」

### 第四課：功能旗標 (Feature Flags)

**目標**：學會使用 `evaluate_feature_flag` 進行受控的功能發布。

**為什麼重要**：功能旗標允許您將程式碼部署與功能發布解耦，從而可以進行灰度發布、A/B 測試，降低新功能上線的風險。

**實踐練習**：

1.  **想像一個新功能**：假設您要為您的應用推出一個「深色模式」的功能。
2.  **設定功能旗標**：在 PostHog 中（或想像中）建立一個名為 `dark_mode` 的功能旗標，並設定為只對 10% 的使用者啟用。
3.  **評估旗標狀態**：使用 Manus 檢查某個特定使用者是否應該看到這個新功能。

    > **指令範例**：「幫我評估一下，使用者 `test-user-alpha` 是否應該啟用 `dark_mode` 這個功能旗標？」

透過完成以上四課的實踐練習，您將具備使用 PostHog 連結器進行基礎產品分析的核心能力。建議您將這些練習應用到您自己的實際產品和數據中，以獲得最深刻的理解。
