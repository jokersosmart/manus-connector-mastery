# 「Google Gemini」連結器使用說明報告

**作者：Joker**

## 簡介

本報告旨在詳細說明 Manus MCP 中的「Google Gemini」連結器。Google Gemini 是一系列功能強大的生成式 AI 模型，能夠理解並處理多種形式的資訊，包括文字、程式碼、圖片等。透過此連結器，使用者可以在 Manus 的自動化工作流程中，無縫地利用 Gemini 的先進能力來處理複雜的任務。

本文件將逐一介紹連結器提供的所有工具、其詳細參數、功能、使用範例及預期輸出，以協助使用者充分發揮 Gemini 連結器的潛力。

## 可用工具概覽

| 工具名稱 | 功能簡述 |
| :--- | :--- |
| `generate_text` | 根據輸入的提示產生文字內容。 |
| `summarize_text` | 將長篇文字內容摘要成精簡版本。 |
| `translate_text` | 將文字從一種語言翻譯成另一種語言。 |
| `generate_code` | 根據需求描述產生程式碼片段。 |
| `analyze_image` | 分析圖片內容並以文字描述。 |

## 工具詳細說明

### 1. generate_text

- **功能說明**: 此工具利用 Gemini 模型，根據使用者提供的文字提示（Prompt）來產生新的文字內容。適用於創意寫作、內容擴寫、草擬郵件、產生文章段落等多種情境。
- **參數**:

| 參數 | 類型 | 是否必須 | 說明 |
| :--- | :--- | :--- | :--- |
| `prompt` | string | 是 | 用於指導模型產生內容的文字提示。 |
| `max_length` | integer | 否 | 產出內容的最大長度（字數或 token 數），預設為 512。 |
| `temperature` | float | 否 | 控制產出內容的隨機性，數值越高（例如 0.8）產出越有創意，數值越低（例如 0.2）產出越穩定且具確定性。預設為 0.7。 |

- **使用範例**:
    - **自然語言指令**: "幫我寫一封電子郵件，邀請我的團隊成員參加下週五下午三點的專案啟動會議。"
    - **工具調用**: `gemini.generate_text(prompt="寫一封電子郵件，邀請團隊成員參加下週五下午三點的專案啟動會議。")`
- **預期輸出**: 一封結構完整、語氣專業的電子郵件草稿。

### 2. summarize_text

- **功能說明**: 此工具能夠將長篇的文章、報告或任何文字段落，自動摘要成一段精簡、易於理解的核心內容。
- **參數**:

| 參數 | 類型 | 是否必須 | 說明 |
| :--- | :--- | :--- | :--- |
| `text` | string | 是 | 需要被摘要的原始文字內容。 |
| `ratio` | float | 否 | 摘要內容相對於原文的長度比例，例如 0.2 代表摘要長度為原文的 20%。預設為 0.2。 |

- **使用範例**:
    - **自然語言指令**: "將這篇文章摘要成 200 字以內。" (並提供文章內容)
    - **工具調用**: `gemini.summarize_text(text="<長篇文章內容>", ratio=0.1)`
- **預期輸出**: 一段簡潔的文字摘要，準確傳達原文的核心思想。

### 3. translate_text

- **功能說明**: 將輸入的文字從來源語言翻譯成指定的目標語言。
- **參數**:

| 參數 | 類型 | 是否必須 | 說明 |
| :--- | :--- | :--- | :--- |
| `text` | string | 是 | 需要翻譯的文字。 |
| `target_language` | string | 是 | 目標語言，例如 "英文"、"繁體中文"、"日文"。 |
| `source_language` | string | 否 | 來源語言。若不指定，模型會自動偵測。 |

- **使用範例**:
    - **自然語言指令**: "把這段話翻譯成英文：'歡迎使用 Manus 的 Google Gemini 連結器。'"
    - **工具調用**: `gemini.translate_text(text="歡迎使用 Manus 的 Google Gemini 連結器。", target_language="英文")`
- **預期輸出**: "Welcome to the Google Gemini connector for Manus."

### 4. generate_code

- **功能說明**: 根據自然語言描述的需求，產生特定程式語言的程式碼片段。
- **參數**:

| 參數 | 類型 | 是否必須 | 說明 |
| :--- | :--- | :--- | :--- |
| `description` | string | 是 | 對所需程式碼功能的自然語言描述。 |
| `language` | string | 是 | 程式語言，例如 "Python"、"JavaScript"、"Go"。 |

- **使用範例**:
    - **自然語言指令**: "用 Python 寫一個函式，可以計算兩個數字的總和。"
    - **工具調用**: `gemini.generate_code(description="一個可以計算兩個數字總和的函式", language="Python")`
- **預期輸出**:

```python
def add_numbers(a, b):
  """This function calculates the sum of two numbers."""
  return a + b
```

### 5. analyze_image

- **功能說明**: 此工具利用 Gemini 的多模態能力，分析輸入的圖片內容，並以文字形式回傳對圖片的描述、辨識出的物件、或回答關於圖片的問題。
- **參數**:

| 參數 | 類型 | 是否必須 | 說明 |
| :--- | :--- | :--- | :--- |
| `image_path` | string | 是 | 圖片的本地檔案路徑或公開的 URL。 |
| `prompt` | string | 否 | 針對圖片提出的具體問題或分析指令。若不提供，則會產生對圖片的通用描述。 |

- **使用範例**:
    - **自然語言指令**: "描述這張圖片裡有什麼。" (並附上圖片)
    - **工具調用**: `gemini.analyze_image(image_path="/path/to/your/image.jpg", prompt="圖片中的主要物件是什麼？")`
- **預期輸出**: 一段描述圖片內容的文字，例如：「圖片中有一隻貓正坐在一張紅色的沙發上。」
