# Perplexity 從新手到精通：您的 AI 研究夥伴養成之路

**作者：Joker**

## 前言：重新定義「問問題」

歡迎來到 Perplexity 的世界。在這裡，您將學會如何將一個簡單的「問題」，轉化為一場深入的「探索」。本指南將帶您從一位對 AI 充滿好奇的新手，一步步成長為能夠駕馭 AI、創造知識的專家。這不僅僅是學習一個工具，更是升級您獲取資訊、進行研究和做出決策的整個操作系統。

準備好了嗎？讓我們開始這趟從新手到精通的旅程。

---

## 等級一：好奇的新手 (The Inquisitive Novice)

**目標：** 建立基本認知，學會提出第一個有價值的問題。

### 核心概念：什麼是答案引擎？

首先，您需要理解 Perplexity 不是傳統的「搜尋引擎」。傳統搜尋引擎給您的是一個「連結列表」，您需要自己點擊、閱讀、比較和總結。而 Perplexity 作為一個「答案引擎」，它會直接給您一個「綜合性的答案」。

> **心法轉變：** 從「找資料」轉變為「問答案」。您不再需要思考要用哪些關鍵字，而是直接用最自然的語言，問出您心中的那個問題。

### 實踐練習：您的第一次提問

忘掉那些用 `+`、`-`、`""` 組合的複雜搜尋語法。現在，像和一位聰明的專家對話一樣，提出您的問題。

**練習指令：**

> 「用 Perplexity 告訴我，什麼是“第一性原理”？」

**Manus 的行動：**

Manus 會將您的自然語言轉換為對 `joker.perplexity.simple_search` 工具的調用。

```json
{
  "tool_name": "joker.perplexity.simple_search",
  "parameters": {
    "query": "What is first principle thinking?"
  }
}
```

**您會得到：** 一段清晰、簡潔的定義，並附上 `[1]`, `[2]` 這樣的引用標記。恭喜，您已經成功邁出了第一步！

## 等級二：資源豐富的學徒 (The Resourceful Apprentice)

**目標：** 學會驗證資訊與聚焦搜尋，確保答案的品質與相關性。

### 核心概念：引用是您的信任之錨

AI 有時會產生「幻覺」。Perplexity 最大的優勢就是提供了**可驗證的引用來源**。您看到的每一個答案，都是從這些來源中提煉總結出來的。學會檢視引用，是從新手進階的關鍵。

> **心法轉變：** 從「相信答案」轉變為「驗證答案」。將 AI 視為一位效率極高的研究助理，但最終的判斷權仍在您手中。

### 實踐練習：聚焦您的研究領域

假設您是一位正在研究再生能源的學生，您不需要全球的資訊，只需要學術界的聲音。這時，`focus` 參數就派上用場了。

**練習指令：**

> 「我想在學術領域裡，用 Perplexity 幫我找找關於“鈣鈦礦太陽能電池”的最新突破。」

**Manus 的行動：**

Manus 會在工具調用中加入 `focus` 參數。

```json
{
  "tool_name": "joker.perplexity.simple_search",
  "parameters": {
    "query": "latest breakthroughs in perovskite solar cells",
    "focus": "academic"
  }
}
```

**您會得到：** 一個專注於學術論文和期刊的答案。現在，試著點擊答案中的 `[1]` 或 `[2]`，親自去閱讀原始文獻，感受一下從「答案」追溯到「證據」的快感。

## 等級三：幹練的研究員 (The Adept Researcher)

**目標：** 掌握深度研究與引導式探索，能夠處理複雜的開放式問題。

### 核心概念：從「問答」到「對話」

真正的研究不是一次性的問答，而是一場持續的對話。當您面對一個複雜主題時，您需要的是一份深度報告，以及下一步該往哪裡探索的建議。這就是 `pro_search` 的價值所在。

> **心法轉變：** 從「被動接收」轉變為「主動引導」。利用 AI 生成的追問建議，層層深入，像剝洋蔥一樣揭開問題的核心。

### 實踐練習：您的第一份 AI 研究報告

您不再滿足於零散的知識點，您想要對一個領域進行系統性的了解。

**練習指令：**

> 「請用 Perplexity 的專業模式，為我深入研究“大型語言模型在藥物研發中的應用”，並告訴我接下來還可以問什麼。」

**Manus 的行動：**

Manus 會調用 `pro_search` 工具，並開啟 `generate_follow_up_questions` 選項。

```json
{
  "tool_name": "joker.perplexity.pro_search",
  "parameters": {
    "query": "Applications of large language models in drug discovery",
    "generate_follow_up_questions": true
  }
}
```

**您會得到：** 一份結構化的報告，包含簡介、多個子主題的分析和結論。更重要的是，您會看到一個「建議追問」的列表。選擇其中一個您最感興趣的問題，開始您的下一輪探索。您已經從一個提問者，進化為一位研究的掌舵者。

## 等級四：戰略大師 (The Strategic Master)

**目標：** 將 Perplexity 無縫整合到您的工作流中，使其成為您值得信賴的「第二大腦」。

### 核心概念：AI 不僅是工具，更是夥伴

到了這個階段，您不再將 Perplexity 視為一個孤立的工具，而是您知識工作流中不可或缺的一環。您可以將 Perplexity 的產出，與 Manus 的其他功能（如文件寫作、數據分析、簡報生成）結合，創造出指數級的價值。

> **心法轉變：** 從「使用工具」轉變為「設計系統」。主動思考如何將 AI 的能力，與您的目標和現有流程相結合。

### 實踐練習：自動化的資訊週報

假設您是一位投資人，需要每週追蹤「半導體產業」的最新動態。

**練習指令（這是一個更複雜的 Manus 任務）：**

1.  **設定排程：** 每週五下午 4 點觸發。
2.  **執行研究：** 使用 `joker.perplexity.pro_search` 搜尋「過去一週半導體產業的重大新聞和技術進展」。
3.  **生成報告：** 將 Perplexity 的輸出，交給 Manus 的寫作工具，整理成一份結構化的 Markdown 週報。
4.  **發送通知：** 將生成的週報發送到您的指定信箱或 Notion 資料庫。

**您會得到：** 一個全自動、客製化的產業情報系統。AI 不再只是被動回答您的問題，而是在主動為您工作，成為您決策的戰略夥伴。

## 超越精通：遠見的建築師 (The Visionary Architect)

達到這個層級，您已經是 Perplexity 的頂尖使用者。您不僅善用工具，更開始思考如何改進工具、拓展邊界。

-   您會向 Manus 團隊反饋關於 Perplexity 連接器的改進建議。
-   您會開始探索將 Perplexity 與其他 MCP 連接器（如 Notion、Cloudflare）組合，創造出前所未有的工作流。
-   您甚至會開始思考，未來的答案引擎應該是什麼樣子，並參與到這個生態的共建之中。

**恭喜您，大師。** 您不僅掌握了一個工具，更掌握了一種與 AI 協作、共同創造知識的未來工作方式。知識的探索永無止境，而您，已經站在了這條路的最前沿。
