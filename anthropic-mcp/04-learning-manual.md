# HubSpot 連接器學習手冊：快速上手指南

本手冊旨在幫助您在最短的時間內掌握使用 HubSpot MCP 連接器的核心技能。我們將透過清晰的學習目標和動手實踐的練習，讓您快速從數據查詢中獲得價值。

作者：Joker

---

## 學習目標

完成本手冊後，您將能夠：

1.  **執行基本的 CRM 數據查詢**：從 HubSpot 中檢索聯絡人、公司和交易等資訊。
2.  **使用篩選條件精準定位數據**：根據特定標準（如金額、日期、狀態）找到您需要的資訊。
3.  **對查詢結果進行排序**：將結果按照您關心的指標（如時間、價值）進行排列。
4.  **獲取特定物件的詳細資料**：深入了解單一聯絡人或交易的完整檔案。

## 關鍵技能培養

### 技能一：您的第一次數據對話

這是您與 HubSpot 數據互動的起點。目標是理解最基本的查詢指令。

- **學習重點**：了解如何向 Manus 提出一個簡單的、開放式的問題來獲取數據。
- **實踐練習**：
  請對 Manus 說出以下任一指令：
  - > 「給我看看最近加入的 3 個聯絡人。」
  - > 「顯示最新的 5 筆交易。」

- **預期成果**：Manus 將返回一個列表，其中包含您所要求的最新 CRM 物件。這證明了您與 HubSpot 之間的通道已經打通。

### 技能二：學會篩選，去蕪存菁

在龐大的數據海洋中，篩選是找到寶藏的地圖。此技能讓您從海量資訊中分離出您真正關心的部分。

- **學習重點**：在您的問題中加入明確的「條件」。
- **實踐練習**：
  嘗試提出一個帶有條件的查詢：
  - > 「找出所有來自『加州』的公司。」
  - > 「列出所有金額超過 8000 美元的交易。」
  - > 「顯示所有處於『已簽約』階段的交易。」

- **預期成果**：您將得到一個經過篩選的、更精簡的列表，其中只包含完全符合您條件的項目。這表示您已學會如何精準定位數據。

### 技能三：排序，讓洞察浮現

數據的價值往往隱藏在順序之中。學會排序，您就能快速識別出「最重要」的資訊。

- **學習重點**：在您的指令中加入「排序」的要求，如「按...排序」、「從高到低」、「從新到舊」。
- **實踐練習**：
  在您的查詢中加入排序指令：
  - > 「列出我們所有的交易，並按金額從高到低排序。」
  - > 「顯示本月建立的所有聯絡人，並按建立日期從新到舊排列。」

- **預期成果**：您看到的結果列表將嚴格按照您指定的順序排列。這讓您可以立即發現最重要的交易或最新的潛在客戶。

### 技能四：深入探索，查看單一實體全貌

有時候，您需要的不只是一個列表，而是某個特定客戶或交易的 360 度視圖。

- **學習重點**：當您知道一個物件的具體 ID 時，如何要求獲取其詳細資訊。
- **實踐練習**：
  （首先，您可以透過之前的查詢找到一個具體的 ID，例如 `deal-12345`）
  然後，對 Manus 說：
  - > 「告訴我關於交易 ID `deal-12345` 的所有資訊。」
  - > 「顯示聯絡人 `contact-98765` 的詳細資料。」

- **預期成果**：Manus 將不再返回列表，而是呈現一個包含該物件所有關鍵屬性（如姓名、電話、郵件、交易金額、狀態等）的詳細卡片。
