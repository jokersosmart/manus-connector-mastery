# Manus MCP Connector Mastery Guide

**作者：Joker**

本儲存庫收錄了 Manus 平台上所有 79 個 MCP（Model Context Protocol）連結器的完整分析文件套件。每個連結器皆包含 5 份深度文件，涵蓋從遞迴分析、使用說明、學習路徑到操作手冊的完整知識體系。

## 專案概覽

| 項目 | 數值 |
| :--- | :--- |
| 連結器總數 | 79 |
| 每個連結器文件數 | 5 |
| 文件總數 | 402 |

## 文件套件結構

每個連結器目錄下包含以下 5 份標準文件：

| 編號 | 檔案名稱 | 說明 |
| :--- | :--- | :--- |
| 01 | `01-recursive-analysis.md` | 4! x 五何法遞迴深度分析報告 |
| 02 | `02-usage-report.md` | 連結器使用說明報告 |
| 03 | `03-from-novice-to-master.md` | 從新手到精通指南 |
| 04 | `04-learning-manual.md` | 學習手冊 |
| 05 | `05-user-manual.md` | 使用手冊 |
