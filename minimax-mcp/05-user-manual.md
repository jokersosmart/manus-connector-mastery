# MiniMax 連結器使用者操作手冊

作者：Joker

## 1. 歡迎使用 MiniMax 連結器

本手冊專為終端使用者設計，旨在幫助您了解如何透過簡單的自然語言指令，來驅動強大的 MiniMax AI 模型完成您的任務。您不需要了解任何程式碼，只需要像與真人助手對話一樣，告訴 Manus 您想做什麼。

## 2. 核心互動模式：從「指令」到「行動」

您與 MiniMax 連結器的互動非常直觀。您用口語化的方式下達指令，Manus 會理解您的意圖，並在幕後將其轉換為對 MiniMax 工具的精確調用。

**您的自然語言指令 → Manus 解讀 → MiniMax 工具調用 → AI 生成結果**

本手冊將透過一系列任務導向的範例，向您展示這個神奇的過程。

## 3. 任務範例：您的日常 AI 應用

### 任務一：快速草擬一份專業郵件

**您的目標：** 您需要發送一封郵件，但不想花時間斟酌字句。

**您的指令：**
> `幫我寫一封郵件給行銷部門的 Sarah，感謝她的團隊上週提供的市場分析報告，並詢問她們是否可以在本週五前，針對報告的第三部分提供更詳細的數據來源。`

**Manus 如何解讀與行動：**

Manus 識別出這是一個「文本生成」任務，核心是創造一段全新的內容。因此，它會選擇 `generate_text` 工具，並將您的具體要求（收件人、感謝內容、問題）打包成一個清晰的 `prompt`。

**幕後工具調用 (示意)：**
```json
{
  "tool": "minimax.generate_text",
  "args": {
    "prompt": "撰寫一封專業的電子郵件。收件人是行銷部門的 Sarah。內容包含：1. 感謝她的團隊上週提供的市場分析報告。2. 詢問是否能在本週五前，提供報告第三部分的詳細數據來源。",
    "temperature": 0.5
  }
}
```

### 任務二：為您的新產品腦力激盪

**您的目標：** 您有一個新產品的初步想法，需要一些創意來豐富它。

**您的指令：**
> `我有一個點子，想做一款專為旅行者設計的智慧筆記本。它應該有什麼樣的特色功能？我們來聊聊。`

**Manus 如何解讀與行動：**

Manus 偵測到「我們來聊聊」這句話，表明這不是一次性的任務，而是一個需要來回溝通的「對話」。因此，它會啟用 `chat` 工具，並將您的第一句話作為對話的開端。

**幕後工具調用 (示意)：**
```json
{
  "tool": "minimax.chat",
  "args": {
    "messages": [
      {
        "role": "user",
        "content": "我有一個點子，想做一款專為旅行者設計的智慧筆記本。它應該有什麼樣的特色功能？我們來聊聊。"
      }
    ]
  }
}
```
當您繼續回覆 AI 的建議時（例如「聽起來不錯，那離線地圖功能可以怎麼整合？」），Manus 會持續將新的對話內容附加到 `messages` 陣列中，讓 AI 始終記得你們的上下文。

### 任務三：將一篇長文變成三分鐘摘要

**您的目標：** 您找到一篇很有價值的網路文章，但沒有時間細讀。

**您的指令：**
> `幫我摘要這篇文章，並列出 5 個重點：[貼上文章連結或全文]`

**Manus 如何解讀與行動：**

這個任務的核心是「理解並濃縮資訊」。Manus 會先使用瀏覽器工具讀取您提供的內容，然後將全文交給 `chat` 或 `generate_text` 工具。它會將「摘要」和「列出 5 個重點」這兩個要求轉換為 `prompt` 的一部分。

**幕後工具調用 (示意)：**
```json
{
  "tool": "minimax.generate_text",
  "args": {
    "prompt": "請將以下文章內容進行摘要，並以無序列表的形式，列出其中最重要的 5 個觀點。\n\n文章內容：...（此處為文章全文）..."
  }
}
```

### 任務四：快速翻譯一份文件

**您的目標：** 您收到一份外文文件，需要快速了解其內容。

**您的指令：**
> `把這段文字翻譯成繁體中文：[貼上外文段落]`

**Manus 如何解讀與行動：**

Manus 從「翻譯成繁體中文」這句話中，精準地捕捉到您的意圖。這同樣是一個「文本生成」任務，只是目標非常明確。

**幕後工具調用 (示意)：**
```json
{
  "tool": "minimax.generate_text",
  "args": {
    "prompt": "Translate the following text to Traditional Chinese:\n\n[貼上外文段落]"
  }
}
```

## 4. 結語：您的想像力是唯一的限制

以上範例僅僅是冰山一角。您可以嘗試組合不同的指令，或者提出更複雜、更具創意的要求。MiniMax 連結器的強大之處，在於它能理解您千變萬化的意圖。現在，就開始用您的語言，指揮您的專屬 AI 吧！
