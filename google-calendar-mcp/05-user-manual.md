# Google Calendar 連結器使用者操作手冊

**作者：Joker**

---

## 簡介

歡迎使用 Google Calendar 連結器！本手冊將向您展示如何透過簡單的自然語言指令，讓 Manus 成為您的智慧排程助理。我們將以任務為導向，提供具體的指令範例，並解釋 Manus 在背後是如何工作的。

## 核心理念：您說話，Manus 執行

與傳統應用程式不同，您不需要學習複雜的介面。您只需要用日常語言告訴 Manus 您的需求。Manus 的人工智慧會分析您的指令，選擇最合適的工具，並填入正確的參數來完成任務。

## 任務一：查看我的日程

這是您了解自己時間安排的最快方式。

| 您可以這樣說... | Manus 是這樣理解的... |
| :--- | :--- |
| 「我明天下午有什麼會議？」 | **意圖**: 查詢日程<br>**工具**: `find_events`<br>**參數**: `start_time` (明天中午12點), `end_time` (明天下午6點) |
| 「幫我看看下週的所有安排。」 | **意圖**: 查詢日程<br>**工具**: `find_events`<br>**參數**: `start_time` (下週一的開始), `end_time` (下週日的結束) |
| 「我今天還有什麼事嗎？」 | **意圖**: 查詢日程<br>**工具**: `find_events`<br>**參數**: `start_time` (現在), `end_time` (今天午夜) |

## 任務三：與他人安排會議

告別繁瑣的來回溝通，讓 Manus 為您找到最佳會議時間。

| 您可以這樣說... | Manus 是這樣理解的... |
| :--- | :--- |
| 「幫我約一下 Joker，主題是『專案進度同步』，明天下午有空嗎？」 | **第一步：檢查空閒時間**<br>**工具**: `check_availability`<br>**參數**: `attendees` (["我", "Joker"]), `start_time` (明天下午), `end_time` (明天下午)<br><br>**第二步：根據您的選擇建立事件**<br>**工具**: `create_event`<br>**參數**: `summary` ("專案進度同步"), `attendees` (["我", "Joker"]), `start_time` (您選擇的時間) |
| 「安排一個會議，邀請 Joker 和 Boss，主題是『年度預算』，下週找個大家都有空的1小時。」 | **意圖**: 尋找空閒時間並建立事件<br>**工具**: `check_availability` -> `create_event`<br>**參數**: `attendees` (["我", "Joker", "Boss"]), `duration_minutes` (60), `time_range` (下週) |

## 常見問題 (FAQ)

**Q: 我需要用非常精確的電腦指令嗎？**

A: 完全不需要！您說得越自然越好。例如，「明天下午三點」和「明天 15:00」對 Manus 來說是一樣的。我們的目標就是讓您忘掉指令，專注於您想做的事。

**Q: 如果 Manus 理解錯了我的意思怎麼辦？**

A: 在執行如建立或修改等關鍵操作前，Manus 通常會向您確認。如果它理解有誤，您只需說「不對，我的意思是...」來修正它。您的回饋會幫助它不斷學習和進步。

**Q: 我的日曆資料安全嗎？**

A: 非常安全。Manus 透過 Google 官方推薦的 OAuth 2.0 協議連接您的帳戶，這是業界最安全的標準。您的資料在傳輸過程中全程加密，而且您隨時可以在您的 Google 帳戶設定中取消對 Manus 的授權。

**Q: 我可以管理多個日曆嗎？**

A: 目前，連結器會預設操作您 Google 帳戶下的主要日曆。在未來的版本中，我們計劃支援讓您指定操作某個特定的子日曆。
