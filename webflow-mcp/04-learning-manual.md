## 學習目標

完成本手冊後，您將能夠：

1.  **檢視您的 Mailchimp 帳戶**：快速列出您所有的受眾（Audiences）。
2.  **管理您的聯絡人**：將新聯絡人加入指定列表，並查詢其資訊。
3.  **操作您的行銷活動**：建立一個行銷活動草稿，並了解如何發送它。
4.  **分析行銷成效**：獲取已發送行銷活動的關鍵表現數據。

## 模組一：檢視您的帳戶 (10 分鐘)

**目標**：學會如何查看您的 Mailchimp 帳戶中有哪些受眾列表。

**關鍵技能**：資訊檢索。

在郵件行銷中，所有聯絡人都儲存在「受眾」中。在進行任何操作前，了解您有哪些受眾是第一步。

### 實踐練習 1：列出我所有的受眾

請對 Manus 說出或輸入以下指令：

> 「用 Mailchimp 幫我列出所有的受眾。」

**Manus 將執行的工具**：`list_audiences`

**您會看到**：一個列表，其中包含您所有受眾的名稱 (`name`) 和它們各自的唯一識別碼 (`id`)。請記下您最常用的那個受眾的 `id`，我們稍後會用到它。

---

## 模組二：管理您的聯絡人 (15 分鐘)

**目標**：學會新增聯絡人到特定受眾，並驗證他們是否成功加入。

**關鍵技能**：資料新增與查詢。

這是連結器最核心的日常應用之一：將新的潛在客戶或訂閱者自動化地加入您的行銷漏斗。

### 實踐練習 2A：新增一個測試聯絡人

現在，請使用您在練習 1 中記下的受眾 `id`，對 Manus 發出指令：

> 「幫我把 `test.user@example.com` 這個聯絡人，以『已訂閱』的狀態，加入到 ID 為 `[請填入您記下的 audience_id]` 的 Mailchimp 受眾中。」

**Manus 將執行的工具**：`add_contact`

**您會看到**：一個確認訊息，表示該聯絡人已被建立。

### 實踐練習 2B：確認聯絡人資訊

為了確保萬無一失，讓我們來檢查一下剛剛加入的聯絡人。

> 「在 ID 為 `[請填入您記下的 audience_id]` 的 Mailchimp 受眾中，幫我找找 `test.user@example.com` 的資訊。」

**Manus 將執行的工具**：`get_contact`

**您會看到**：該聯絡人的詳細資料，包括其 email 地址、訂閱狀態等，確認一切都如您預期。

## 模組三：操作您的行銷活動 (15 分鐘)

**目標**：學會如何建立一個行銷活動的草稿。

**關鍵技能**：執行建立與發送操作。

學會了管理聯絡人，下一步就是向他們發送內容。

### 實踐練習 3：建立一個行銷活動草稿

讓我們來建立一個準備發送給您主要受眾的活動。

> 「幫我用 Mailchimp 建立一個新的常規行銷活動，目標受眾是 ID 為 `[請填入您記下的 audience_id]` 的列表，郵件主旨是『我的第一個自動化行銷活動』，寄件人名稱是『[您的名字]』。」

**Manus 將執行的工具**：`create_campaign`

**您會看到**：一個新的行銷活動被建立，狀態為 `draft`（草稿）。您會獲得一個新的 `campaign_id`。記下這個 ID！

*注意：透過 API 建立的行銷活動預設沒有內容。您需要登入 Mailchimp 網站來編輯郵件的視覺設計和文字。這個工具主要用於快速啟動一個活動的框架。*

---

## 模組四：分析行銷成效 (10 分鐘)

**目標**：學會如何查詢一個已發送行銷活動的表現。

**關鍵技能**：數據分析與解讀。

了解哪些有效、哪些無效，是優化未來行銷策略的關鍵。

### 實踐練習 4：獲取活動報告

找一個您過去已經發送過的行銷活動來進行分析。如果您不確定 ID，可以先用 `list_campaigns` 並將狀態設為 `sent` 來查找。

> 「幫我獲取 ID 為 `[請填入一個已發送的 campaign_id]` 的 Mailchimp 行銷活動的成效報告。」

**Manus 將執行的工具**：`get_campaign_report`

**您會看到**：一份詳細的報告，包含開信率 (`open_rate`)、點擊率 (`click_rate`) 等關鍵數據。這些指標是衡量您的郵件內容是否吸引人的重要依據。
