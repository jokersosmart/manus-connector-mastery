# Stripe API 連結器學習手冊

作者：Joker

## 簡介：專為行動而設計

歡迎使用 Stripe API 連結器學習手冊。與傳統的長篇大論不同，本手冊專為「行動」和「快速上手」而設計。我們相信，學習新工具最好的方式就是直接使用它來解決真實問題。

本手冊將圍繞一系列明確的「學習目標」來組織，每個目標都對應一個或多個您可以立即上手的「實踐練習」。我們將省略繁瑣的理論，直擊核心，讓您在最短的時間內掌握使用此連結器提升工作效率的關鍵技能。

我們的核心理念是**「場景抽象」**。您將學會如何用自然語言描述您想完成的支付任務，而不是去記憶複雜的 API 端點和參數。準備好開始了嗎？

## Part 1: 核心收款技能

### 學習目標 1：在 5 分鐘內建立一個可收款的頁面

這是使用本連結器最基礎、也是最核心的技能。掌握它，意味著您擁有了將任何想法快速變現的能力。

*   **核心工具：** `create_payment_link`
*   **關鍵概念：** 將「產品」、「價格」、「支付頁面」三合一的場景抽象。

#### 實踐練習

1.  **為固定價格產品收款：**
    *   **任務：** 您想銷售一本 29 美元的電子書《Growth Hacking 101》。
    *   **指令：** `"Create a payment link for 'Growth Hacking 101' at $29."`
    *   **驗證：** 點擊 Manus 回傳的 URL，確認您能看到一個包含正確產品名稱和價格的 Stripe Checkout 頁面。

2.  **為服務項目收款（允許調整數量）：**
    *   **任務：** 您提供每小時 100 美元的設計諮詢服務，客戶可能一次購買多個小時。
    *   **指令：** `"Generate a payment link for 'Design Consulting Hour' at 100 USD, and make the quantity adjustable."`
    *   **驗證：** 在打開的支付頁面上，確認您可以看到一個可以修改數量的輸入框。

### 學習目標 2：有效管理您的客戶資料

客戶是您業務的基礎。學會如何快速查詢和管理他們，是日常營運的必備技能。

*   **核心工具：** `create_customer`, `list_customers`
*   **關鍵概念：** 將客戶視為一個獨立的資源，進行增、查操作。

#### 實踐練習

1.  **手動新增潛在客戶：**
    *   **任務：** 您在一次會議上認識了 `Big Corp` 的採購經理 `Tom Chen`，他的郵箱是 `tom.chen@bigcorp.com`。您希望將他加入系統以便後續跟進。
    *   **指令：** `"Add a new customer: name 'Tom Chen (Big Corp)', email 'tom.chen@bigcorp.com'."`
    *   **驗證：** 接著使用下一個練習中的指令，確認 `Tom Chen` 已成功加入。

2.  **快速查找特定客戶：**
    *   **任務：** 您需要查找 `Tom Chen` 的客戶記錄。
    *   **指令：** `"Search for the customer with the email 'tom.chen@bigcorp.com'."`
    *   **驗證：** 確認 Manus 回傳了 `Tom Chen` 的客戶資料。
