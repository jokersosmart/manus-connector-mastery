# Typeform 連結器學習手冊

**作者：Joker**

## 簡介

本手冊旨在幫助您快速上手 Manus 的 Typeform 連結器，並掌握其核心技能。透過清晰的學習目標和實踐練習，您將能夠在短時間內將 Typeform 的資料收集能力與 Manus 的自動化能力結合，提升您的工作效率。

## 學習目標

完成本手冊後，您將能夠：

1.  **成功連結**您的 Typeform 帳戶與 Manus。
2.  **熟練查詢**您帳戶中的表單與回應。
3.  **獨立建立**新的 Typeform 表單。
4.  **設計並執行**一個基礎的自動化工作流程。

## 關鍵技能與實踐練習

### 技能一：帳戶連結與基本查詢

這是使用連結器的第一步，也是最重要的基礎。

- **學習重點：**
  - 了解 OAuth 2.0 授權流程。
  - 使用 `list_forms` 工具檢視帳戶中的所有表單。
  - 使用 `get_responses` 工具檢視指定表單的回應。

- **實踐練習：**
  1.  **練習 1-1：** 首次使用 Typeform 連結器，並按照提示完成帳戶授權。
  2.  **練習 1-2：** 向 Manus 下達指令：「列出我 Typeform 帳戶裡最近修改的 5 個表單。」
  3.  **練習 1-3：** 選擇您的一個表單，向 Manus 下達指令：「幫我看看 [表單名稱] 這個 Typeform 表單的總共有多少筆回應？」

### 技能二：動態表單建立

掌握此技能，您將能讓 Manus 成為您快速產生表單的得力助手。

- **學習重點：**
  - 了解 `create_form` 工具的必要參數：`title` 和 `fields`。
  - 學習如何用自然語言描述您想要的表單欄位結構。

- **實踐練習：**
  1.  **練習 2-1：** 構思一個簡單的「午餐意見調查」表單，包含「姓名」和「今天想吃什麼？」(選擇題) 兩個欄位。
  2.  **練習 2-2：** 向 Manus 下達指令：「幫我用 Typeform 建立一個『午餐意見調查』，要有『姓名』欄位，和一個選項是『中餐』、『西餐』、『日式料理』的選擇題。」
  3.  **練習 2-3：** 建立一個包含「電子郵件」和「手機號碼」欄位的「聯絡資訊收集」表單。

### 技能三：進階回應篩選

學會精準地篩選回應，是高效處理大量資料的關鍵。

- **學習重點：**
  - 掌握 `get_responses` 工具的 `since`、`until` 和 `completed` 參數。
  - 學習如何組合使用這些參數來縮小查詢範圍。

- **實踐練習：**
  1.  **練習 3-1：** 找出您某個表單在「昨天」一整天內收到的所有「已完成」的回應。
  2.  **練習 3-2：** 向 Manus 下達指令：「幫我篩選出 [表單名稱] 在上週收到的所有回應。」

### 技能四：基礎自動化流程設計

將上述技能串連起來，解決一個真實世界的問題。

- **學習重點：**
  - 學習如何將一個任務拆解成多個步驟。
  - 思考如何將一個工具的輸出作為下一個工具的輸入。

- **實踐練習：**
  1.  **練習 4-1：** 設計一個流程，手動觸發後，可以將您某個 Typeform 表單的「所有回應」轉換成一個 Markdown 格式的表格。
      - **提示：** 第一步使用 `get_responses`，第二步讓 Manus 進行資料格式轉換。

## 結語

學習的最好方式就是不斷地動手嘗試。請務必完成以上所有實踐練習，並鼓勵您根據自己的實際需求，設計更多、更複雜的挑戰。當您遇到問題時，別忘了 Manus 的社群和文件永遠是您最好的後盾。祝您學習愉快！
