# Airtable 連結器使用者操作手冊

**作者：Joker**

---

## 1. 簡介：讓 Airtable 聽懂你的話

歡迎使用 Airtable 連結器！本手冊將向你展示如何透過簡單的自然語言指令，讓 Manus 成為你強大的 Airtable 助理。你不需要了解複雜的程式碼或 API，只需要像平常說話一樣，告訴 Manus 你想做什麼。

本手冊以任務為導向，為你提供具體的指令範例，並揭示 Manus 在背後是如何理解你的意圖，並將其轉換為精確的工具調用。準備好，讓你的資料管理工作變得前所未有的簡單高效。

## 2. 核心任務：查詢資料

這是你最常用的功能：從你的資料庫中找到你需要的資訊。

**你的目標**：從一個名為「專案追蹤」的 Base 中，找到「任務」表格裡所有正在進行中的任務，並且希望它們按照截止日期的先後順序排列。

| 你可以這樣說 | Manus 的內心戲 (工具調用) |
| :--- | :--- |
| 「幫我從『專案追蹤』的『任務』表格裡，找出所有狀態是『進行中』的任務，並按截止日期升序排序。」 | **工具**：`list_records`<br>**參數**：<br>- `base_id`: `appXXXXXXXXXXXXXX`<br>- `table_id`: `tblXXXXXXXXXXXXXX`<br>- `filter_by_formula`: `"{Status} = '進行中'"`<br>- `sort`: `[{"field": "截止日期", "direction": "asc"}]` |

**解讀**：當你提到「找出」、「列出」或「篩選」，Manus 就會準備使用 `list_records` 工具。它會根據你提到的「狀態是進行中」來設定 `filter_by_formula`，並根據「按截止日期升序排序」來設定 `sort` 參數。

## 3. 核心任務：新增資料

當你有新的資訊需要記錄下來時，直接告訴 Manus 即可。

**你的目標**：在你的「客戶管理」Base 的「聯絡人」表格中，快速新增一位潛在客戶的資料。

| 你可以這樣說 | Manus 的內心戲 (工具調用) |
| :--- | :--- |
| 「在『聯絡人』表格裡加一筆新資料：姓名是『王五』，公司是『未來科技』，電話是『0987-654-321』。」 | **工具**：`create_record`<br>**參數**：<br>- `base_id`: `appYYYYYYYYYYYYYY`<br>- `table_id`: `tblYYYYYYYYYYYYYY`<br>- `records`: `[{"fields": {"姓名": "王五", "公司": "未來科技", "電話": "0987-654-321"}}]` |

**解讀**：當你說「新增」、「加入」或「建立一筆」，Manus 就會啟用 `create_record` 工具。它會聰明地將你提到的「姓名」、「公司」、「電話」等資訊，對應到 `records` 物件中的 `fields`，完成資料的新增。

## 4. 核心任務：修改資料

更新資訊的狀態或內容，是日常工作的一部分。

**你的目標**：你剛剛完成了一項任務，需要將其在 Airtable 中的狀態更新為「已完成」。你知道這項任務的紀錄 ID 是 `recABC123456`。

| 你可以這樣說 | Manus 的內心戲 (工具調用) |
| :--- | :--- |
| 「把 ID 是 `recABC123456` 的任務狀態改成『已完成』。」 | **工具**：`update_record`<br>**參數**：<br>- `base_id`: `appXXXXXXXXXXXXXX`<br>- `table_id`: `tblXXXXXXXXXXXXXX`<br>- `records`: `[{"id": "recABC123456", "fields": {"Status": "已完成"}}]` |

**解讀**：指令中的「更新」、「修改」或「改成」會觸發 `update_record` 工具。Manus 會利用你提供的 `ID` 來定位要修改的紀錄，並將你指定的欄位 (`Status`) 更新為新的內容 (`已完成`)。

## 5. 核心任務：刪除資料

當某些資訊不再需要時，你可以讓 Manus 幫你清理。

**你的目標**：從「待辦事項」表格中，刪除一條已經取消的任務，其紀錄 ID 為 `recXYZ987654`。

| 你可以這樣說 | Manus 的內心戲 (工具調用) |
| :--- | :--- |
| 「幫我把 ID 是 `recXYZ987654` 的紀錄刪掉。」 | **工具**：`delete_record`<br>**參數**：<br>- `base_id`: `appXXXXXXXXXXXXXX`<br>- `table_id`: `tblXXXXXXXXXXXXXX`<br>- `record_ids`: `["recXYZ987654"]` |

**解讀**：「刪除」、「移除」或「丟掉」這類詞語會讓 Manus 選擇 `delete_record` 工具。它只需要你提供要刪除紀錄的 `ID`，就能精準地完成操作。

## 結語：你的專屬資料助理

正如你所見，與 Airtable 的互動可以像聊天一樣簡單。你只需要專注於你想「做什麼」，而 Manus 會負責處理「如何做」。

現在就開始嘗試用你自己的話，向 Manus 下達指令吧！你會驚訝地發現，管理資料從未如此直觀和輕鬆。
