## 1. 核心工具集

### 1.1 `get_quote` (獲取即時行情)

此工具，是您的「市場報價機」，提供指定標的的最新、最即時的交易數據。

*   **用途**：快速查詢當前價格、日內波動、以及即時的買賣盤口。
*   **命令結構**：
    ```json
    {
      "ticker": "TICKER_SYMBOL"
    }
    ```
*   **參數詳解**：

| 參數 | 類型 | 是否必須 | 描述與範例 |
| :--- | :--- | :--- | :--- |
| `ticker` | `string` | 是 | 您要查詢的金融標的代碼。**必須使用標準格式**。<br>股票: `AAPL`<br>指數: `I:SPX`<br>外匯: `C:EUR-USD`<br>加密貨幣: `X:BTC-USD` |

*   **操作範例**：
    *   **您的指令**：「`現在比特幣的價格是多少？`」
    *   **Manus 內部執行的 JSON 輸入**：`{ "ticker": "X:BTC-USD" }`

### 1.2 `get_aggregates` (獲取歷史 K 線數據)

此工具，是您的「時間機器」，可以回溯任何一個標的在過去任意時間段的價格走勢。

*   **用途**：進行技術分析、研究歷史趨勢、回測交易策略。
*   **命令結構**：
    ```json
    {
      "ticker": "TICKER_SYMBOL",
      "from": "YYYY-MM-DD",
      "to": "YYYY-MM-DD",
      "timespan": "day | hour | minute"
    }
    ```
*   **參數詳解**：

| 參數 | 類型 | 是否必須 | 描述與範例 |
| :--- | :--- | :--- | :--- |
| `ticker` | `string` | 是 | 要查詢的金融標的代碼。 |
| `from` | `string` | 是 | 開始日期，格式為 `YYYY-MM-DD`。 |
| `to` | `string` | 是 | 結束日期，格式為 `YYYY-MM-DD`。 |
| `timespan` | `string` | 是 | K 線的時間週期。可選值包括 `minute`, `hour`, `day`, `week`, `month`, `quarter`, `year`。 |

*   **操作範例**：
    *   **您的指令**：「`我想看看特斯拉從 2022 年 1 月 1 日到 2023 年 12 月 31 日的週線圖。`」
    *   **Manus 內部執行的 JSON 輸入**：`{ "ticker": "TSLA", "from": "2022-01-01", "to": "2023-12-31", "timespan": "week" }`

### 1.3 `get_news` (獲取相關新聞)

此工具，是您的「市場情報員」，為您收集與特定公司或整個市場相關的最新資訊。

*   **用途**：理解股價波動背後的原因、追蹤行業動態、監控競爭對手。
*   **命令結構**：
    ```json
    {
      "ticker": "TICKER_SYMBOL",
      "limit": 50
    }
    ```
*   **參數詳解**：

| 參數 | 類型 | 是否必須 | 描述與範例 |
| :--- | :--- | :--- | :--- |
| `ticker` | `string` | 否 | 如果提供，則返回與該特定標的相關的新聞。如果省略，則返回市場的普遍性新聞。 |
| `limit` | `integer` | 否 | 返回的新聞條目數量上限。預設為 50，最大可根據您的權限設定。 |

*   **操作範例**：
    *   **您的指令**：「`最近有什麼關於晶片行業的新聞嗎？給我最新的 5 條。`」
    *   **Manus 內部執行的 JSON 輸入**：`{ "ticker": "NVDA", "limit": 5 }` (Manus 可能會選擇一個代表性的 Ticker 如 NVDA 來查詢行業新聞)

### 1.4 `get_financials` (獲取標準化財報)

此工具，是您的「X 光透視儀」，可以深入剖析一家上市公司的財務健康狀況。

*   **用途**：進行基本面分析、公司估值、財務建模。
*   **命令結構**：
    ```json
    {
      "ticker": "TICKER_SYMBOL",
      "timeframe": "annual | quarterly"
    }
    ```
*   **參數詳解**：

| 參數 | 類型 | 是否必須 | 描述與範例 |
| :--- | :--- | :--- | :--- |
| `ticker` | `string` | 是 | 要查詢的上市公司代碼。 |
| `timeframe` | `string` | 是 | 您想要的財報週期。`annual` 代表年報，`quarterly` 代表季報。 |

*   **操作範例**：
    *   **您的指令**：「`請給我微軟最新的季度財報。`」
    *   **Manus 內部執行的 JSON 輸入**：`{ "ticker": "MSFT", "timeframe": "quarterly" }`

### 1.5 `get_ticker_details` (獲取公司基本資訊)

此工具，是公司的「數字名片」，提供了關於一家公司的核心背景資訊。

*   **用途**：快速了解一家不熟悉的公司、獲取行業分類、查找官方網站。
*   **命令結構**：
    ```json
    {
      "ticker": "TICKER_SYMBOL"
    }
    ```
*   **參數詳解**：

| 參數 | 類型 | 是否必須 | 描述與範例 |
| :--- | :--- | :--- | :--- |
| `ticker` | `string` | 是 | 要查詢的上市公司代碼。 |

*   **操作範例**：
    *   **您的指令**：「`亞馬遜是做什麼的？`」
    *   **Manus 內部執行的 JSON 輸入**：`{ "ticker": "AMZN" }`

## 2. 組合使用範例

**場景**：一位分析師，想要快速評估一家公司（例如 `SNOW` - Snowflake Inc.）的財報後表現。

1.  **第一步：查看財報後的價格反應**
    *   **指令**：「`請給我 SNOW 在財報發布日（假設是 2023-11-29）當天及後續 5 個交易日的每小時 K 線圖。`」
    *   **Manus 執行**：`get_aggregates("SNOW", from="2023-11-29", to="2023-12-06", timespan="hour")`，並繪製圖表。

2.  **第二步：尋找相關新聞解釋**
    *   **指令**：「`財報發布前後，市場上有哪些關於 SNOW 的重要新聞？`」
    *   **Manus 執行**：`get_news("SNOW", limit=10)`，並重點呈現財報發布日附近的新聞。

3.  **第三步：深入財報細節**
    *   **指令**：「`現在，把 SNOW 該季度的財報調出來，我需要看他們的收入增長和客戶數。`」
    *   **Manus 執行**：`get_financials("SNOW", timeframe="quarterly")`，並利用 LLM，從返回的數據中，提取並總結關於「收入」和「客戶指標」的關鍵資訊。

通過這一系列連貫的操作，分析師，可以在幾分鐘內，完成一個小型、但數據充分的財報後分析報告。
