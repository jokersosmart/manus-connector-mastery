# Anthropic 連結器使用者操作手冊

**作者：Joker**

歡迎使用 Anthropic 連結器！本手冊專為終端使用者設計，旨在向您展示如何透過簡單的自然語言指令，釋放 Anthropic Claude AI 模型的強大潛力。您將看到，您無需了解複雜的程式碼或 API 細節，只需像與真人對話一樣，即可完成各種任務。

## 核心理念：從「指令」到「行動」

在 Manus 平台，您與 Anthropic 連結器的互動非常直觀。您用自然的語言描述您想做什麼，Manus 的智慧核心會自動將您的話語翻譯成對 Anthropic 連結器 `create_message` 工具的精確調用。

以下我們將透過一系列任務導向的範例，揭示這一神奇過程的幕後工作原理。

--- 

### 任務一：快速獲取資訊

**您的目標**：您想了解一個新概念「量子計算」。

**您的指令**：
> `「請用簡單的語言解釋一下什麼是量子計算。」`

**Manus 的幕後工作**：

Manus 接收到您的指令後，會構建一個如下的工具調用請求，並發送給 Anthropic 連結器：

```json
{
  "tool": "anthropic.create_message",
  "parameters": {
    "model": "claude-3-sonnet-20240229",
    "messages": [
      {
        "role": "user",
        "content": "請用簡單的語言解釋一下什麼是量子計算。"
      }
    ],
    "max_tokens": 500
  }
}
```

**解析**：這是一個最直接的轉換。您的問題被直接放入 `messages` 陣列的 `content` 欄位，Manus 選擇了一個性能均衡的 `sonnet` 模型來回答您的問題。

--- 

### 任務二：總結一篇長篇報告

**您的目標**：您有一篇非常長的市場分析報告，沒有時間細讀，需要快速掌握其核心內容。

**您的指令**：
> `「這是一份關於 2025 年電動車市場的報告，請幫我總結出 5 個最重要的趨勢，並列點說明。報告內容如下：...（此處貼上萬字報告全文）...」`

**Manus 的幕後工作**：

Manus 識別出您需要處理長文本並進行總結，它會選擇最強大的模型來確保分析品質：

```json
{
  "tool": "anthropic.create_message",
  "parameters": {
    "model": "claude-3-opus-20240229",
    "system": "你是一位資深的市場分析師，請精準地總結文本的核心要點。",
    "messages": [
      {
        "role": "user",
        "content": "這是一份關於 2025 年電動車市場的報告，請幫我總結出 5 個最重要的趨勢，並列點說明。報告內容如下：...（報告全文）..."
      }
    ],
    "max_tokens": 800
  }
}
```

**解析**：
*   **模型選擇**：由於任務複雜且文本長，Manus 自動升級到能力最強的 `opus` 模型。
*   **系統提示**：Manus 為 AI 設定了「資深市場分析師」的角色 (`system` 提示)，以確保總結的專業性和準確性。
*   **長文本處理**：您的長篇報告全文被完整地傳遞給模型，發揮了 Anthropic 領先的長文本處理能力。

--- 

### 任務三：草擬一封專業的道歉郵件

**您的目標**：您需要就服務中斷向一位重要客戶致歉，但希望語氣誠懇、專業且不失溫度。

**您的指令**：
> `「幫我草擬一封給『未來公司』的張總的道歉信，內容是關於我們昨天的系統意外中斷了 2 小時。強調我們已經修復了問題，並將提供一個月的免費服務作為補償。我的職位是客戶成功經理，王景理。」`

**Manus 的幕後工作**：

Manus 會將您的具體要求細化，並注入到系統提示中，以精確控制郵件的風格和內容。

```json
{
  "tool": "anthropic.create_message",
  "parameters": {
    "model": "claude-3-sonnet-20240229",
    "system": "你是一位經驗豐富的客戶成功經理，名叫王景理。你的寫作風格專業、誠懇且富有同理心。現在你需要撰寫一封道歉郵件。",
    "messages": [
      {
        "role": "user",
        "content": "收件人是『未來公司』的張總。事件是昨天的系統意外中斷 2 小時。解決方案是問題已修復，並提供一個月免費服務作為補償。請基於以上資訊撰寫郵件。"
      }
    ],
    "max_tokens": 400
  }
}
```

**解析**：Manus 將您的指令拆解為「角色設定」和「具體任務資訊」。角色設定被放入 `system` 提示，而具體的郵件要素則作為 `content` 傳遞。這種分離使得 AI 能更好地代入角色，同時又不會遺漏關鍵資訊。

--- 

### 任務四：讓 AI 成為您的程式設計夥伴

**您的目標**：您想用 Python 寫一個簡單的網路爬蟲，但不太熟悉相關的函式庫。

**您的指令**：
> `「請用 Python 和 BeautifulSoup 函式庫寫一個程式，它可以抓取 `https://example.com` 這個網站的標題 (title) 並印出來。」`

**Manus 的幕後工作**：

Manus 識別出這是一個程式碼生成任務，會引導模型提供清晰、可執行的程式碼。

```json
{
  "tool": "anthropic.create_message",
  "parameters": {
    "model": "claude-3-opus-20240229",
    "system": "你是一位精通 Python 的資深軟體工程師，擅長編寫簡潔、高效且帶有清晰註解的程式碼。",
    "messages": [
      {
        "role": "user",
        "content": "請用 Python 和 BeautifulSoup 函式庫寫一個程式，它可以抓取 `https://example.com` 這個網站的標題 (title) 並印出來。"
      }
    ],
    "max_tokens": 600
  }
}
```

**解析**：同樣地，Manus 為 AI 設定了「資深軟體工程師」的專家角色，並選用了邏輯推理能力更強的 `opus` 模型，以確保生成程式碼的品質和正確性。

## 結語

正如您所見，您與 Anthropic 連結器的互動是如此簡單而強大。您只需專注於清晰地表達您的「目標」，剩下的繁瑣工作——選擇哪個模型、如何設定參數、如何構建請求——Manus 都會為您智慧地處理。

現在，就去嘗試下達您的第一個指令吧！探索、實驗，您會驚訝地發現，這位強大的 AI 夥伴能為您的工作和生活帶來多大的改變。
