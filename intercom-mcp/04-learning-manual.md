# 「Linear」連結器學習手冊

**作者：Joker**

---

本學習手冊旨在幫助您快速上手 Manus MCP「Linear」連結器，並掌握其關鍵技能。我們將透過清晰的學習目標和實踐練習，讓您在最短的時間內發揮此工具的最大潛力。

## 學習目標

完成本手冊的學習後，您將能夠：

*   獨立使用自然語言指令，透過 Manus Agent 完成 Linear 中的日常任務。
*   理解 Manus 指令與 Linear API 工具調用之間的對應關係。
*   組合不同的工具，建立自動化的專案管理工作流程。

## 第一章：快速上手

### 學習目標

*   成功設定並驗證 Linear 連結器的連線。
*   執行您的第一個查詢指令，並理解其回傳結果。
*   執行您的第一個變更指令，並在 Linear 介面中確認變更。

### 實踐練習 1：您的第一個指令

1.  **目標:** 查詢您在 Linear 上的所有團隊。
2.  **指令:** 在 Manus Agent 中輸入：「`列出我在 Linear 上的所有團隊。`」
3.  **驗證:** 檢查 Manus Agent 回傳的結果是否包含您所屬團隊的正確資訊 (ID, Name, Key)。

### 實踐練習 2：建立您的第一個議題

1.  **目標:** 在您的其中一個團隊中，建立一個測試議題。
2.  **指令:** 在 Manus Agent 中輸入：「`在 [您的團隊名稱] 團隊建立一個標題為「我的第一個 Manus 議題」的議題。`」 (請將 `[您的團隊名稱]` 替換為您真實的團隊名稱或 Key)。
3.  **驗證:** 前往 Linear 的網頁介面，確認該議題是否已成功建立在對應的團隊中。
