# JSONBin.io 連結器學習手冊

**作者：** Joker

---

本手冊旨在幫助您快速上手 JSONBin.io 連結器，並掌握其核心關鍵技能。我們將透過清晰的學習目標和具體的實踐練習，讓您在最短的時間內將 JSONBin.io 應用於您的專案中。

## 學習目標

完成本手冊後，您將能夠：

1.  **理解核心概念：** 獨立說明什麼是 Bin 和 Collection。
2.  **掌握基本操作：** 熟練使用連結器對單個 JSON 資料 (Bin) 進行建立、讀取、更新和刪除 (CRUD)。
3.  **學會組織資料：** 使用 Collection 來管理和組織多個 Bins。
4.  **應用於實踐：** 將 JSONBin.io 作為一個簡易的後端資料來源，應用到一個小型專案中。

## 模組一：您的第一個雲端 JSON (約 15 分鐘)

**目標：** 了解 JSONBin.io 的基本單位「Bin」，並成功建立您的第一筆資料。

### 關鍵知識

- **Bin：** 在 JSONBin.io 中，一個 Bin 就是一筆獨立的 JSON 資料。您可以把它看作一個存在雲端的 `.json` 檔案。
- **公開 vs. 私有：** 建立 Bin 時，您可以選擇將其設定為「公開」（任何人知道 ID 就能讀取）或「私有」（只有您自己能讀取），這是最基本的權限控制。

### 實踐練習

請使用 `create_bin` 工具完成以下任務：

1.  **建立您的個人設定檔：**
    -   **內容：** `{"name": "Your Name", "goal": "Learn JSONBin.io"}`
    -   **設定：** 將其設定為 **私有 (private)**。
    -   **記錄：** 保存返回結果中的 `bin_id`，我們稍後會用到它。

2.  **建立一個公開的筆記：**
    -   **內容：** `{"note": "Hello World from JSONBin.io!"}`
    -   **設定：** 保持其預設的 **公開 (public)** 狀態。

## 模組二：讓資料井然有序 (約 20 分鐘)

**目標：** 學會使用「Collection」來組織和歸類您的 Bins。

### 關鍵知識

- **Collection：** 一個 Collection 就像一個資料夾，它可以包含多個 Bins。當您的 Bins 數量變多時，使用 Collection 可以讓資料管理變得更加輕鬆。

### 實踐練習

1.  **建立一個專案資料夾：**
    -   使用 `create_collection` 工具建立一個名為 `My-Todo-List` 的集合。
    -   記錄返回結果中的 `collection_id`。

2.  **將待辦事項放入資料夾：**
    -   使用 `create_bin` 工具建立兩個新的 Bins，代表您的待辦事項。
    -   **Bin 1 內容：** `{"task": "Finish learning manual", "done": false}`
    -   **Bin 2 內容：** `{"task": "Build a cool app", "done": false}`
    -   在建立這兩個 Bins 時，務必使用 `collection_id` 參數，將它們歸類到您剛剛建立的 `My-Todo-List` 集合中。

3.  **檢視您的資料夾：**
    -   使用 `list_bins_in_collection` 工具，並傳入您的 `collection_id`，查看 `My-Todo-List` 集合中是否包含了您剛剛建立的兩個待辦事項。

## 模組三：日常資料維護 (約 15 分鐘)

**目標：** 熟練掌握對現有資料的讀、改、刪操作。

### 關鍵知識

- **CRUD：** 這是資料庫操作的四個基本動作：Create (建立)、Read (讀取)、Update (更新)、Delete (刪除)。您已經學會了 Create，現在來完成剩下的部分。

### 實踐練習

請針對您在模組二中建立的待辦事項 Bin 進行以下操作：

1.  **讀取資料：**
    -   使用 `read_bin` 工具，並傳入第一個待辦事項的 `bin_id`，來讀取其詳細內容。

2.  **更新狀態：**
    -   假設您已經完成了第一個任務，現在使用 `update_bin` 工具來更新它。
    -   **新的內容：** `{"task": "Finish learning manual", "done": true}`
    -   **提示：** 更新是 **完全覆蓋**，所以您需要傳入完整的 JSON 內容，而不僅僅是 `"done": true`。

3.  **刪除記錄：**
    -   假設您決定放棄第二個任務，使用 `delete_bin` 工具將其從您的待辦清單中永久移除。

## 總結與下一步

恭喜！您已經掌握了 JSONBin.io 連結器的所有核心技能。您現在可以獨立地使用它來儲存、組織和管理您的 JSON 資料了。

**下一步挑戰：**

嘗試將今天所學應用到一個您自己的小專案中。例如，您可以建立一個簡單的個人書籤管理器，用一個 Collection 來存放所有書籤，每個書籤都是一個 Bin。這將是鞏固所學知識的最佳方式。
