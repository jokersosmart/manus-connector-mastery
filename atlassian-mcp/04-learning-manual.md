# Atlassian 連接器學習手冊

作者：Joker

## 簡介

本手冊旨在幫助您快速上手 Manus Atlassian 連接器，並掌握其核心技能。我們將透過一系列清晰的學習目標和實踐練習，讓您在最短的時間內，體驗到自動化為您的 Atlassian 工作流程帶來的改變。

## 學習目標

完成本手冊的學習後，您將能夠：

1.  **理解 Atlassian 連接器的核心價值**：明白它為何能解決您在 Jira 和 Confluence 操作中的痛點。
2.  **獨立完成基本的 Jira 任務自動化**：能夠使用自然語言在 Jira 中建立、搜尋議題。
3.  **獨立完成基本的 Confluence 頁面操作**：能夠使用自然語言在 Confluence 中建立、更新頁面。
4.  **執行一次完整的 Jira 到 Confluence 的手動同步流程**：體驗連接器如何簡化跨產品操作。

## 關鍵技能培養

### 技能一：用自然語言下達指令

這是使用 Manus 的最核心技能。您需要學會將您的「操作意圖」轉換為清晰、無歧義的自然語言指令。一個好的指令通常包含以下元素：

*   **平台 (Platform):** 您想在哪個軟體中操作？(例如：Jira, Confluence)
*   **動作 (Action):** 您想做什麼？(例如：建立、搜尋、更新)
*   **目標 (Target):** 您想操作哪個物件？(例如：任務、頁面、專案)
*   **參數 (Parameters):** 執行動作所需的具體資訊。(例如：標題、內容、專案鍵)

--- 

### 實踐練習 1：建立您的第一個 Jira 任務

*   **目標：** 在 Jira 中建立一個任務。
*   **場景：** 您需要為自己安排一個學習任務。
*   **指令：** 請對 Manus 說出以下或類似的指令：
    > 「在 Jira 的 `PER` 專案裡，幫我開一個新任務，標題是『完成 Atlassian 連接器學習手冊』，把它指派給我自己。」
*   **驗證：** 前往您的 Jira `PER` 專案，看看是否成功建立了這個任務。

--- 

### 技能二：理解並利用 ID 與 Key

在 Atlassian 的世界中，幾乎所有的物件（專案、議題、頁面）都有一個獨一無二的 ID 或 Key。學會利用這些識別碼，可以讓您的指令更加精確。

*   **Jira Project Key:** 每個專案的縮寫，例如 `PROJ`。
*   **Jira Issue Key:** 專案 Key 加上一個數字，例如 `PROJ-123`。
*   **Confluence Page ID:** 一串數字，例如 `12345678`。

當您不確定頁面或任務的完整名稱時，使用 ID 或 Key 是最可靠的方式。

--- 

### 實踐練習 2：更新一個 Confluence 頁面

*   **目標：** 更新一個已存在的 Confluence 頁面。
*   **場景：** 您需要在一份會議記錄頁面中，補充一些新的討論內容。
*   **前置作業：** 請先在您的 Confluence 中手動建立一個頁面，並取得它的 Page ID。（您可以在頁面 URL 的尾端找到它，或者點擊頁面右上角的「...」 -> 「頁面資訊」中找到）
*   **指令：** 假設您的 Page ID 是 `98765`，請對 Manus 說：
    > 「幫我更新 Confluence 頁面 `98765`，在頁面最下方加上一行字：『補充：已確認由 Joker 負責後續追蹤。』」
*   **驗證：** 重新整理該 Confluence 頁面，確認內容是否已成功更新。

--- 

### 技能三：組合指令，簡化流程

自動化的精髓在於將多個步驟串連起來。雖然目前我們還需要手動分步執行，但這能幫助您理解未來「規則引擎」的運作邏輯。

--- 

### 實踐練習 3：從 Jira 到 Confluence 的手動同步

*   **目標：** 將 Jira 的任務更新，同步到 Confluence 的專案日誌中。
*   **場景：** 您剛剛在 Jira 中完成了一個重要的 Bug 修復，現在需要在團隊的專案日誌中記錄下來。
*   **步驟：**
    1.  **找出 Bug 資訊：**
        > 「幫我看一下 Jira 任務 `BUG-42` 的標題是什麼？」
        *   *(Manus 返回：「BUG-42 的標題是『修復使用者無法登入的問題』」)*
    2.  **更新 Confluence 日誌：**
        > 「在 Confluence 頁面 `Project Log` (ID: 556677) 中，新增一行：『[已完成] 修復使用者無法登入的問題 (BUG-42)。』」
*   **驗證：** 前往 Confluence 的 `Project Log` 頁面，確認這筆更新已經被記錄下來。
