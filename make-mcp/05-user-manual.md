# Google Gemini 連結器使用者操作手冊

**作者：Joker**

## 歡迎使用 Gemini 連結器

本手冊旨在為終端使用者提供一份清晰、易懂的操作指南。與其關注技術細節，我們將聚焦於您在日常工作中可能遇到的具體任務，並展示如何透過簡單的自然語言指令，讓 Manus 協調 Google Gemini 的強大 AI 為您完成工作。

核心理念是：**您只管下達指令，剩下的交給 Manus。**

在本手冊中，我們將透過一系列「任務場景」來展示這一點。每個場景都會包含：

-   **您的任務**：一個具體的、您想要完成的目標。
-   **您的指令**：您需要對 Manus 說的自然語言指令。
-   **Manus 的幕後工作**：解釋 Manus 如何理解您的意圖，並將其轉換為對 Gemini 工具的精確調用。

## 任務場景一：快速草擬一份會議邀請

-   **您的任務**：您需要邀請您的團隊成員參加一個關於新專案的腦力激盪會議。
-   **您的指令**：
    > 「幫我寫一封會議邀請郵件，主題是『新專案 Alpha 腦力激盪會』，時間是下週三下午 2 點，地點是 301 會議室。請大家準備好自己的想法。」
-   **Manus 的幕後工作**：
    1.  **意圖識別**：Manus 偵測到您需要「撰寫郵件」，這是一個典型的文字生成任務。
    2.  **工具選擇**：它決定使用 `gemini.generate_text` 工具。
    3.  **參數轉換**：Manus 會將您指令中的所有關鍵資訊（主題、時間、地點、會議目的）整合到一個結構化的提示（Prompt）中，然後傳遞給 Gemini。這個提示可能看起來像這樣：`prompt="撰寫一封會議邀請郵件，主題為『新專案 Alpha 腦力激盪會』，會議時間為下週三下午 2 點，地點在 301 會議室。郵件內容需提及這是一場腦力激盪會議，並請與會者預先準備想法。"`
    4.  **結果呈現**：Gemini 根據這個詳細的提示，生成一封格式完整、內容清晰的郵件草稿，直接呈現給您。

## 任務場景二：從海量資訊中提取重點

-   **您的任務**：您拿到一份長達 20 頁的市場研究報告，但您只有 5 分鐘時間來了解其核心內容。
-   **您的指令**：
    > 「將這份報告摘要成 5 個重點條列。」（同時將報告檔案或文字貼給 Manus）
-   **Manus 的幕後工作**：
    1.  **意圖識別**：Manus 識別出「摘要」的需求。
    2.  **工具選擇**：它選擇了 `gemini.summarize_text` 工具。
    3.  **參數轉換**：Manus 將您提供的整個報告內容作為 `text` 參數，並理解您需要一個條列式的、包含 5 個重點的摘要。它可能會在內部對提示進行優化，例如：`prompt="請將以下文字摘要成 5 個最重要的核心重點，並以條列式呈現。"`
    4.  **結果呈現**：Gemini 閱讀並分析整份報告，然後回傳給您 5 條最關鍵的資訊摘要。

## 任務場景三：讓程式碼為您工作

-   **您的任務**：您正在處理一些數據，需要一個簡單的 Python 腳本來幫您從一個列表中篩選出所有大於 100 的數字。
-   **您的指令**：
    > 「用 Python 寫一個函式，它可以接收一個數字列表，並回傳一個只包含其中大於 100 的數字的新列表。」
-   **Manus 的幕後工作**：
    1.  **意圖識別**：Manus 偵測到您需要「編寫程式碼」。
    2.  **工具選擇**：它選擇了 `gemini.generate_code` 工具。
    3.  **參數轉換**：您的指令被直接轉換為 `description` 參數，同時 Manus 會將 `language` 參數設為「Python」。
    4.  **結果呈現**：Gemini 根據您的功能描述，生成一段乾淨、高效且包含註解的 Python 程式碼，您可以直接複製並在您的專案中使用。

## 任務場景四：看圖說故事

-   **您的任務**：您在網路上看到一張有趣的圖片，想用它來發一篇社群貼文，但不知道該如何描述。
-   **您的指令**：
    > 「幫我描述一下這張圖片，並為它想一個有趣的標題。」（同時附上該圖片）
-   **Manus 的幕後工作**：
    1.  **意圖識別**：Manus 理解您需要對圖片進行「描述」和「創意發想」。這是一個多步驟的任務。
    2.  **工具鏈（Tool Chaining）**：Manus 會設計一個兩步驟的工作流程：
        a.  **第一步：分析圖片**。它會先調用 `gemini.analyze_image` 工具，將圖片路徑作為輸入，以獲得對圖片內容的基礎文字描述。
        b.  **第二步：創意寫作**。接著，它會調用 `gemini.generate_text` 工具，將第一步得到的圖片描述作為提示的一部分，並結合您的要求（「想一個有趣的標題」）來生成最終的社群貼文內容。
    3.  **結果呈現**：您會得到一段生動的圖片描述和一個或多個有趣的標題選項，可以直接用於您的社群媒體。

## 結論

如您所見，您無需了解 `prompt`、`temperature` 或 `tool chaining` 這些複雜的概念。您只需要用最自然的方式，告訴 Manus 您想做什麼。Manus 和 Gemini 會在幕後處理所有複雜的技術細節，將強大的 AI 能力轉化為您指尖的生產力工具。
