# 「LINE」連結器學習手冊

作者：Joker

## 簡介

本手冊旨在幫助您快速上手 Manus 的「LINE」連結器，掌握其核心功能，並立即將其應用於您的自動化工作流程中。我們將專注於最關鍵的技能，並提供清晰的學習目標和實踐練習。

## 模組一：發送您的第一則 LINE 訊息

**學習目標：** 了解 `send_message` 工具的基本用法，並成功透過 Manus 發送一則 LINE 訊息。

**關鍵技能：**

-   **指定接收者 (`recipient_id`)：** 每個 LINE 使用者、群組或聊天室都有一個獨一無二的 ID。您必須提供這個 ID，才能將訊息發送到正確的地方。
-   **撰寫訊息內容 (`message`)：** 您想要發送的文字內容。

**實踐練習：**

1.  **獲取您的 LINE 使用者 ID：** 您可以透過 LINE Developers 後台或特定的 LINE 機器人來查詢您的個人使用者 ID (格式通常為 `U` 開頭)。
2.  **執行您的第一個指令：**

    > **自然語言指令：**
    > 「用 LINE 傳送訊息給 `U123456...`，內容是『這是我用 Manus 發的第一則 LINE 訊息！』」

**成功標準：** 您的 LINE 帳號成功收到由 Manus 發送的訊息。

## 模組二：建立一個簡單的回覆機器人

**學習目標：** 了解如何使用 `reply_to_message` 工具來回應來自使用者的訊息。

**關鍵技能：**

-   **理解 `reply_token`：** 當使用者傳訊息給您的 LINE 官方帳號時，LINE 會透過一個 Webhook 事件通知您。該事件中包含一個一次性的 `reply_token`，您必須使用這個 token 才能回覆該則訊息。
-   **設定 Webhook：** 您需要在 LINE Developers 後台設定一個 Webhook URL，這是一個公開的網路端點，用來接收 LINE 發送的事件通知。

**實踐練習：**

1.  **設定您的 LINE 官方帳號與 Webhook。**
2.  **建立一個 Manus 工作流程：** 設定一個觸發器，當您的 Webhook URL 收到 POST 請求時，解析出 `reply_token` 和使用者訊息。
3.  **執行回覆指令：**

    > **自然語言指令 (在您的工作流程腳本中觸發)：**
    > 「使用 LINE 回覆訊息，使用權杖 `nHuyWiB7y...`，內容是『我收到你的訊息了！』」

**成功標準：** 當您向您的 LINE 官方帳號發送訊息時，會立即收到自動回覆。

## 模組三：整合與應用

**學習目標：** 學習將 LINE 連結器與其他工具結合，解決一個實際問題。

**關鍵技能：**

-   **流程設計：** 思考如何將多個工具串連起來，完成一個比單一工具更複雜的任務。
-   **變數傳遞：** 將一個工具的輸出（例如，從 Google Sheets 讀取的數據）作為另一個工具（例如，LINE `send_message`）的輸入。

**實踐練習：**

-   **任務：** 建立一個每日天氣提醒機器人。
-   **步驟：**
    1.  使用 `search` 工具或專門的天氣 API 獲取您所在城市今日的天氣預報。
    2.  將天氣資訊整理成一段文字。
    3.  使用 LINE `send_message` 工具，將這段天氣預報發送給您自己。
    4.  (進階) 設定一個排程觸發器，讓這個流程在每天早上 8 點自動執行。

**成功標準：** 您每天早上都能準時收到個人化的天氣預報 LINE 訊息。
