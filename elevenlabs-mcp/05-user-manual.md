# ElevenLabs 終端使用者操作手冊

**作者：Joker**

---

## 簡介

歡迎使用 ElevenLabs MCP 連結器！本手冊專為終端使用者設計，旨在向您展示如何透過簡單的自然語言指令，讓 Manus 為您完成高品質的文字轉語音任務。您不需要了解複雜的程式碼或 API 細節，只需像和助理對話一樣，告訴 Manus 您想要什麼。

## 核心概念：從「指令」到「執行」

您與 Manus 的互動是基於「意圖」的。您用日常語言描述您的目標，Manus 則負責將其「翻譯」成對 ElevenLabs 工具的精確調用。以下是一些關鍵的對應關係，這將幫助您更有效地提出指令：

*   **您的指令中提到「聲音」** (例如「用 Rachel 的聲音」)，Manus 會去尋找對應的 `voice_id`。
*   **您的指令中提到「內容」** (例如「幫我念一段話...」)，Manus 會將這段話填入 `text` 參數。
*   **您的指令中描述「風格」** (例如「用活潑一點的語氣」、「聽起來專業一些」)，Manus 會智慧地調整 `voice_settings` 中的 `stability` 等參數。

## 任務導向指令範例

讓我們透過一系列具體的任務，來看看您該如何下達指令。

### 任務 1：快速生成一段語音

這是最基礎也最常見的需求。您有一段文字，希望快速將它轉換成語音。

**情境：** 您想將一句歡迎詞轉換成語音。

| 您可以這樣說 (自然語言指令) | Manus 會這樣理解 (工具調用) |
| :--- | :--- |
| 「使用 ElevenLabs 幫我生成一句話：『歡迎來到我們的產品發布會！』」 | `{
  "tool_name": "text_to_speech",
  "parameters": {
    "text": "歡迎來到我們的產品發布會！"
  }
}` |
| 「用 ElevenLabs 念一下這段文字：『...』」 | (同上) |

*   **提示：** 在未指定聲音時，Manus 會使用預設聲音。

### 任務 2：指定特定的配音員

ElevenLabs 擁有豐富的聲音庫，您可以像挑選演員一樣，選擇最適合您內容的聲音。

**情境：** 您正在製作一個兒童故事，希望用一個聽起來友善、溫和的聲音 (例如「Charlotte」) 來敘述。

| 您可以這樣說 (自然語言指令) | Manus 會這樣理解 (工具調用) |
| :--- | :--- |
| 「用 ElevenLabs 的 Charlotte 聲音，幫我講一個故事：『很久很久以前，有一隻愛吃蜂蜜的小熊...』」 | `{
  "tool_name": "text_to_speech",
  "parameters": {
    "voice_id": "XB0fDUnXU5powFXDhCwa",
    "text": "很久很久以前，有一隻愛吃蜂蜜的小熊..."
  }
}` |
| 「我想要 Charlotte 的聲音，幫我念這段話...」 | (同上) |

### 任務 3：導演聲音的情感與風格

您不僅是編劇，更是導演。您可以指導聲音的情感表現，讓它更貼合您的內容氛圍。

**情境 A：** 您需要為一則嚴肅的新聞報導配音。

| 您可以這樣說 (自然語言指令) | Manus 會這樣理解 (工具調用) |
| :--- | :--- |
| 「用 ElevenLabs 的 James 聲音，以非常專業、穩定的新聞播報風格，念出這段頭條：『本市宣布將於下月啟動新的環保計畫。』」 | `{
  "tool_name": "text_to_speech",
  "parameters": {
    "voice_id": "...",
    "text": "...",
    "voice_settings": {
      "stability": 0.8
    }
  }
}` |

**情境 B：** 您需要為一個充滿激情的產品廣告配音。

| 您可以這樣說 (自然語言指令) | Manus 會這樣理解 (工具調用) |
| :--- | :--- |
| 「我需要一個充滿活力和激情的聲音！用 ElevenLabs 的 Chris 聲音，幫我念這句廣告詞：『立即行動，改變您的未來！』」 | `{
  "tool_name": "text_to_speech",
  "parameters": {
    "voice_id": "...",
    "text": "...",
    "voice_settings": {
      "stability": 0.4
    }
  }
}` |

*   **提示：** Manus 會根據您使用的形容詞（如「穩定」、「專業」、「活力」、「激情」）來自動調整 `stability` 參數。

### 任務 4：為長篇內容（如播客、有聲書）生成旁白

當處理長篇文本時，為了讓語氣過渡更自然，您可以向 Manus 提供上下文資訊。

**情境：** 您正在製作有聲書的第二章，希望它的語氣能與第一章結尾和第三章開頭順暢地銜接。

| 您可以這樣說 (自然語言指令) | Manus 會這樣理解 (工具調用) |
| :--- | :--- |
| 「我要用 ElevenLabs 生成一段語音。**這是要生成的內容：**『他推開了那扇沉重的門...』。**它的前一段是：**『...他終於找到了傳說中的城堡。』**後一段是：**『門後的世界讓他驚呆了。』」 | `{
  "tool_name": "text_to_speech",
  "parameters": {
    "text": "他推開了那扇沉重的門...",
    "previous_text": "...他終於找到了傳說中的城堡。",
    "next_text": "門後的世界讓他驚呆了。"
  }
}` |

*   **提示：** 使用「前一段是...」、「後一段是...」這樣的句式，可以最清晰地告訴 Manus 如何使用上下文。

## 結語

與 ElevenLabs 連結器的互動，就像與一位專業的配音助理合作。您只需清晰地表達您的創作意圖，剩下的技術細節就交給 Manus 來處理。現在就開始嘗試吧，用您的聲音，創造無限可能！
